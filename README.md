# Dudo
Dudo (Spanish for I doubt), also known as Cacho, Pico, Perudo, Liar's Dice, Cachito or Dadinho is a popular dice game played in South America

# How to run
Debug mode:
```sh
(cd client && npm install)
(cd server && npm install)
make debug
```
Release mode:
```sh
(cd client && npm install)
(cd server && npm install)
export Port=5000
export HOST=$(e.g. YOUR_IP)
make release
```
